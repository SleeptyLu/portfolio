AchieveTrack

Um agregador centralizado para platinadores, focado em análise de dados e consolidação de conquistas multi-plataforma.

Vanilla JS Tailwind CSS REST APIs Build: v0.1-alpha

A Necessidade

Como platinadora, percebi que o meu esforço está fragmentado. Tenho conquistas na Steam, troféus na PSN e conquistas na Xbox, mas não existe um local que unifique o meu valor total como jogadora de forma elegante e analítica.

O AchieveTrack nasce para resolver isto, criando um dashboard único onde os dados são os protagonistas.

Funcionalidades Chave

  • ▹ Integração com Steam Web API para leitura em tempo real.
  • ▹ Cálculo de "Raridade Média" de platinas por conta.
  • ▹ Gráficos de progressão mensal de jogos concluídos.

Arquitetura do Projeto

O foco deste website é a performance. Quero que a análise seja instantânea e que a interface responda como uma aplicação nativa.

Explorar Stack Técnica Fechar Detalhes

Lógica de Dados

Utilizo JavaScript assíncrono (fetch/promises) para comunicar com as APIs. O desafio técnico aqui é tratar os dados JSON brutos e transformá-los em métricas visuais compreensíveis através de lógica de arrays e objetos.

Design Adaptável

Com o Tailwind CSS, criei um sistema de design baseado em tokens de cores que permite uma interface dark-mode agressiva e profissional, focada na legibilidade dos números e estatísticas.
